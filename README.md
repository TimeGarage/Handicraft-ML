# Handicraft-ML
### 前言

为了加深对机器学习经典算法的理解，手动实现一遍机器学习经典算法。同时，对于每一种算法，在博客中总结相应的心得体会，希望能对学习者有所帮助。

**本项目参考书籍：李航《统计学习方法》**

**我的博客：[「少数派报告」](http://www.timegarage.works)**
